# X (Twitter) Post Format

Write a single X post (≤280 characters) announcing the most user-visible approved change.
Optionally, when several distinct features ship, follow it with a short thread (2–4 posts),
each its own line block separated by a blank line and prefixed `2/`, `3/`, etc.

Rules:

1. Lead with the user-facing benefit in the first line — what someone can now do.
2. Plain language. No corporate throat-clearing ("We're thrilled to announce…").
3. At most one or two hashtags, and only if genuinely relevant. No emoji walls.
4. End with the release link placeholder if a link is provided; never invent a URL.
5. No fabricated metrics, customer names, unreleased promises, or availability dates —
   only what the approved evidence supports.

Keep the lead post tight enough to read at a glance. The founder voice (tone, vocabulary)
comes from the brand-voice skill layered on top of this format.
