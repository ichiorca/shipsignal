# Brand Voice

Write in a clear, confident, and helpful voice. Favor concrete user value over hype.

- Lead with what the user can now do, not with internal implementation detail.
- Prefer plain language; avoid jargon, acronyms, and superlatives ("revolutionary",
  "best-in-class") unless they are backed by approved evidence.
- Never invent metrics, percentages, customer names, or roadmap promises.
- Keep sentences short. Use active voice. Address the reader as "you".
